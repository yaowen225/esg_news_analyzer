# ESG分析評估指引：Reflection Agent參考手冊

## 1. 評估標題列表質量

### 高質量標題列表的特徵
- **完整性**：應達到10條不同的標題，若少於10條需有合理解釋
- **多樣性**：標題應涵蓋E（環境）、S（社會）和G（治理）三個ESG主要面向
- **時效性**：標題應為近期新聞，反映當前趨勢
- **清晰度**：標題應簡潔明了，不含多餘符號或混亂格式

### 需要修正的情況
- 標題數量明顯不足（少於7條）且無合理解釋
- 標題高度重複或極其相似
- 標題格式混亂，含有多餘前綴/後綴或HTML殘留
- 標題不是新聞標題，而是其他網頁元素（如導航欄文字）

## 2. 評估趨勢分析質量

### 高質量趨勢分析的特徵
- **分類完整**：明確將標題歸類至E、S、G三大類別，並提供數量統計
- **數據支持**：使用具體數字/百分比支持觀點，如「環境相關標題佔60%」
- **深度洞察**：超越表面描述，指出標題背後的核心趨勢和發展方向
- **全球視角**：考慮地域分布和影響，特別是標題中提及的國家/地區
- **行業關聯**：識別涉及的特定行業和部門，顯示跨行業的ESG趨勢

### 需要修正的情況
- 分析缺乏基本的E、S、G分類
- 純粹描述性，缺乏數據支持或量化分析
- 分析中出現與標題無關的猜測或外部信息
- 存在明顯的事實錯誤或邏輯矛盾
- 漏掉標題中明顯的關鍵趨勢

## 3. 評估影響分析的充分程度

評估之前的分析預期會產生什麼影響，這些影響分析是否考慮以下因素：

### 關鍵評估維度
- **全面性**：是否從企業、投資者、社會和監管多角度分析
- **連貫性**：影響分析是否與前面的趨勢分析邏輯一致
- **平衡性**：是否同時考慮了正面和負面潛在影響
- **實用性**：分析是否提供了實際可行的洞見，而非空泛表述
- **深度**：是否探討了直接影響之外的連鎖或長期效應

## 4. 反思輸出指引

### 為高質量分析提供的反思
當分析整體良好時，提供這些反思點以增強draft_agent的影響分析：
```markdown
我認為analyst_agent的分析整體上是全面且有洞見的。以下是我的評估和反思洞見：

[具體評估意見，肯定分析的優點]

額外影響分析考量：
- [提供1-2個缺少的潛在影響]
- [提供1-2個未被深入討論的利益相關者角度]
- [提供1-2個可能被忽略的風險或機遇]

評估與反思完成
```

### 需要修正時的輸出模板
當發現嚴重問題需要修正時，使用以下模板：
```markdown
需要修正

我發現以下問題需要解決：
1. [具體問題描述]
2. [如有多個問題，逐一列出]

這些問題影響了分析的[準確性/完整性/可信度]，建議進行修正。
```

## 5. 優質影響分析案例與要素

### 案例分析
以下是一個高質量影響分析的示例節錄：

**對企業的影響**:
*深度*：不僅提到運營成本降低，還提及品牌形象提升和風險減少
*平衡性*：同時考慮機會（成本降低）和挑戰（需要資訊透明度）
*具體性*：提供具體機制解釋，如「減少信息不對稱導致的風險」

### 常見問題與改進方向
- **過於籠統**：避免籠統表述如「將產生重大影響」，應說明具體是什麼影響及其機制
- **僅關注短期**：應同時考慮短期和長期影響
- **單一維度**：避免只關注經濟或環境層面，應考慮社會、技術等多層面
- **過於樂觀**：應平衡討論潛在挑戰和風險
- **缺乏機制解釋**：不僅指出結果，還應解釋「為什麼」和「如何」

## 6. ESG趨勢評估的黃金標準

### 近期全球ESG關鍵趨勢
- 氣候變遷相關披露與減碳承諾的標準化（尤其是範疇3排放）
- 自然資本和生物多樣性的價值融入財務決策
- ESG報告框架整合與簡化
- 供應鏈盡職調查與透明度
- 可持續金融產品的擴展與標準化
- 循環經濟模式採用增加
- 能源轉型加速（可再生能源、儲能技術）
- 社會公平與包容性指標納入ESG評估

比對分析師的趨勢識別與這些全球趨勢的一致性，但也應認可新興趨勢的識別。
